Obtaining an export certificate using Apply for an export certificate: step by step

An explanation of how to go about obtaining an export certificate using this service.

  1. Step 1: Check if you need a certificate

    Use Apply for an export certificate to check if you need a certificate based on destination country, certificate type and goods type. Read the resulting guidance to understand what you need to do.

    1. Check if you need a certificate

  2. Step 2: Create an account

    This account will allow you to make an access request. The access request will allow you to apply for an export certificate.

    1. Create an account

  3. Step 3: Apply for an export certificate

    You must create an account before you can apply for a certificate.

    Once your account is created and your access request is approved, you will be able to sign into your account to apply, manage existing applications or renew certificates.

    1. Sign into your account
    2. Help using Apply for an export certificate

  4. and Await issuance, rejection or additional information request

    You will receive an approval, a detailed explanation for a rejection or a request for additional information.

# data-hub-prototypes

## Prerequisites

- NodeJS - preferably installed using [nvm](https://github.com/nvm-sh/nvm)

Refer to the node value in `.nvmrc`

```
nvm install 16.14.0
```

Run locally

```
npm install
npm start
```

- [GOV.UK prototype kit developer docs](https://govuk-prototype-kit.herokuapp.com/docs)
